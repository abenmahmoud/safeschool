# SafeSchool 🛡️

Application web anti-harcèlement scolaire.

## Fonctionnalités
- 📢 **Signaler** — Décrire une situation de harcèlement anonymement
- 📚 **Apprendre** — Comprendre le harcèlement, quiz interactif
- 🤖 **Assistant Léa** — Parler, être écouté(e)
- 🆘 **Aide** — Contacts & urgences (3018, 15 SAMU)

## Déploiement
Déployé sur Netlify : https://darling-muffin-21eb90.netlify.app

## Architecture
- Frontend : HTML/CSS/JS (single file)
- Hébergement : Netlify
- CI/CD : GitHub → Netlify (automatique)
